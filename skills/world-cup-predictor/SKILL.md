# 2026 FIFA世界杯预测器

48支球队、12个小组、104场比赛。主办国：美国/加拿大/墨西哥。

## 使用时机

触发："预测2026世界杯" / "世界杯预测" / "谁能夺冠" / "对阵图" / "金靴奖" / "分析巴西vs阿根廷" / "更新预测" / "世界杯黑马"

**不适用于**：女子世界杯、俱乐部世界杯、历史回顾、非2026赛事。

## 执行流程

### 模式: full（全量预测）

#### 第1阶段：数据采集（web_search，6-10次）

| # | 查询 | 目标 |
|---|------|------|
| 1 | `FIFA world ranking 2026 latest` | 48队最新排名 |
| 2 | `2026 World Cup qualified teams complete list` | 确认参赛球队（注意已淘汰的球队如意大利） |
| 3 | `2026 World Cup latest news today` | **当日最新新闻（必做，确保时效性）** |
| 4 | `2026 World Cup predictions odds favorites` | 专家预测和赔率 |
| 5 | `World Cup 2026 injury tracker ruled out [当前月份] 2026` | 已确认缺席/存疑名单（必做） |
| 5b | `[球星名] injury update [月份] 2026` | 前8球队核心球员伤情（按需） |
| 6 | `[热门球队] friendly results 2026` | **近期友谊赛/预选赛结果（评估实际状态）** |
| 7 | `[球队名] 2026 recent results goals` | 评分关键球队补充数据 |

> **时效性要求**：搜索必须使用当前月份作为时间限定，优先获取最近1周/1月的结果。避免使用过时数据。

#### 第2阶段：球队评分

8维加权评分（满分100），详见 `references/scoring-weights.md`：

```
近期状态×20% + FIFA排名×15% + 历史世界杯×15% + 核心球员×12%
+ 进攻×10% + 防守×10% + 场地条件×10% + 战术体系×8%
```

边界：卫冕冠军历史分-2 | 主办国场地满分+2 | 淘汰赛经验+3/-2

#### 第2.5阶段：关键球员状态修正（重要！）

根据实时伤病数据修正评分（规则详见 `references/scoring-weights.md` 下半部分）：

```
修正流程：伤病名单(缺席/存疑/带伤/健康) → 按球员等级(S/A/B/C)计算修正
→ 评估替补深度 → 累积放大(2人×1.2, 3+人×1.5) → 最终评分=基础+修正
```

**报告必须包含**：调整前vs调整后对照表、每队伤病影响分析、排名变动说明。

#### 第3阶段：小组赛模拟

12组逐一模拟：组内4队两两对阵→按评分差预测结果/比分→计算积分→排名1-4位→汇总第三名取前8晋级。

#### 第4阶段：淘汰赛预测

32强对阵图（参考 historical-world-cup-data.md）→ 应用压力系数 → 逐轮预测至决赛。

#### 第5阶段：奖项预测

- **金靴**：进攻核心进球效率 × 出场次数（与晋级路径一致）
- **金球**：四强核心球员综合能力
- **金手套**：四强门将零封率
- **最佳新秀**：U21球员表现和影响力

#### 第6阶段：生成报告

保存到 `WorldCup2026_Prediction_YYYYMMDD.md`，终端输出执行摘要。

### 其他模式

- **match**：搜索两队数据 → 评分 → 胜平负概率 + 预测比分 + 关键对位
- **group**：搜索4队数据 → 评分 → 模拟6场 → 排名表
- **champion**：精简采集(搜索1-4) → 前16评分 → 冠军/奖项预测
- **update**：读取旧报告 → 重新采集 → 标注"↑↓→"变化 → 新报告

## 注意事项

- **数据**：中英文双语搜索，优先FIFA/ESPN/转会市场，交叉验证，TBD标记待定球队
- **质量**：小组赛6场互相影响不孤立预测，淘汰赛考虑路径难度，奖项与晋级路径一致
- **输出**：全中文，比分"X:Y"，中文队名附英文，报告存项目根目录
- **局限**：统计模型有不确定性，仅供娱乐，切勿用于赌博
