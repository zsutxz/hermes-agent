## 2026世界杯新赛制规则

### 基本赛制
- 48支球队分为12个小组（A-L），每组4队
- 总计104场比赛
- 小组赛：每队3场比赛（组内单循环）

### 晋级规则
- 每组前2名直接晋级（共24队）
- 8个成绩最好的小组第三名晋级（共8队）
- 合计32支球队进入淘汰赛

### 32强对阵规则

淘汰赛首轮（32强）的对阵编排如下：

```
场次1:  1A vs 2B          场次9:  1A vs 2B 胜者相关路径
场次2:  1C vs 2D          场次10: 1C vs 2D 胜者相关路径
场次3:  1E vs 2F          场次11: 1E vs 2F 胜者相关路径
场次4:  1G vs 2H          场次12: 1G vs 2H 胜者相关路径
场次5:  1I vs 2J          场次13: 1I vs 2J 胜者相关路径
场次6:  1K vs 2L          场次14: 1K vs 2L 胜者相关路径
场次7:  1B vs 2A          场次15: 1B vs 2A 胜者相关路径
场次8:  1D vs 2C          场次16: 1D vs 2C 胜者相关路径
```

**第三名球队对阵规则**（根据FIFA官方赛制）：

小组第三名按排名(1-8)分配到以下位置：

| 第三名排名 | 对阵 | 对手位置 |
|-----------|------|----------|
| 第1 | 替代位置 | 取决于具体抽签 |
| 第2 | 替代位置 | 取决于具体抽签 |
| ... | ... | ... |

**重要说明**：32强中，8场为小组第一vs小组第二的固定对阵，另外8场涉及小组第三名球队。第三名球队的对阵取决于其所属小组及最终排名，具体分配由FIFA预设的赛程表决定。执行预测时需确保：
1. 先完成所有12组的小组赛预测
2. 排名8个最佳第三名
3. 按照FIFA预设赛程将第三名分配到对应位置
4. 生成完整的32强对阵图

### 淘汰赛路径
```
32强 → 16强 → 八强 → 半决赛 → 季军赛/决赛
 16场    8场    4场    2场      2场
```

## 卫冕冠军"魔咒"

| 年份 | 卫冕冠军 | 上届冠军 | 下届成绩 |
|------|----------|----------|----------|
| 2022 | 法国 | 2018冠军 | 亚军（点球负） |
| 2018 | 德国 | 2014冠军 | 小组赛出局 |
| 2014 | 西班牙 | 2010冠军 | 小组赛出局 |
| 2010 | 意大利 | 2006冠军 | 小组赛出局 |
| 2006 | 巴西 | 2002冠军 | 八强 |
| 2002 | 法国 | 1998冠军 | 小组赛出局 |

**规律**：自1998年以来，卫冕冠军在下届世界杯中仅有法国(2022)进入决赛，其余多在小组赛或早期淘汰赛出局。此因素应在评分算法中作为减分项考虑。
