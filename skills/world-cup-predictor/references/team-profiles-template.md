# 球队数据采集模板

## 基本信息
```
球队名称 / 足协 / FIFA排名 / 是否已确定参赛
```

## 竞技数据
```
历史世界杯参赛次数 / 最佳成绩 / 上届(2022)成绩
近12个月战绩：胜 平 负 / 场均进球 / 场均失球
近5场国际赛结果：(日期 对手 比分)
```

## 核心球员（含状态评估）
```
球星1：姓名 / 俱乐部 / 位置 / 等级(S/A/B/C) / 状态(健康/轻微/存疑/带伤/缺席)
球星2：同上格式
球星3：同上格式
替补深度：极深/良好/一般/薄弱
```

## 伤病追踪（每次执行时更新）
```
确认缺席：球员名 / 伤病类型 / 替代方案
存疑出战：球员名 / 伤病类型 / 出场概率
非伤病风险：转会低水平联赛 / 更衣室矛盾 / 长期缺赛复出
```

## 战术与条件
```
主教练 / 常用阵型 / 战术风格 / 战术成熟度
是否主办国 / 气候适应 / 旅途距离
```

## 搜索查询（每队5-6次）
1. `[队名] FIFA ranking 2026` — 排名
2. `[队名] 2025 2026 国际赛结果` — 近期战绩
3. `[队名] 世界杯历史成绩` — 历史数据
4. `[队名] 关键球员 伤病 2026` — 球员状态
5. `[队名] 主教练 战术 2026` — 战术信息
6. `[球星名] injury update [月份] 2026` — 具体伤情（按需）

## 数据优先级
1. **必须**：FIFA排名、近期战绩、历史世界杯成绩
2. **尽量**：核心球员状态、伤病、战术体系
3. **可选**：详细比分、训练情况

未确定球队标记为"待定(TBD)"，用预选赛排名估算。
